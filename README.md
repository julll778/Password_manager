# Password Manager
A secure password manager built with Python and PyQt6

## Features
- Secure password storage with Fernet encryption
- Password strength validation
- Password generator
- BCrypt password hashing

## Technologies Used
- Python 3.13
- PyQt6
- SQLite3
- Cryptography
- BCrypt

## How to Run
1. Clone the repo
2. pip install -r requirements.txt
3. python main.py
